## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 🎬 **Cinematic Preloader** | Animated loading screen with school logo |
| 🖱️ **Custom Interactive Cursor** | Dynamic trailing dot with hover expand effects |
| 📊 **Scroll Progress Bar** | Glowing top bar tracking scroll position |
| 🔢 **Animated Statistics** | Counter animations for Students, Teachers, Years |
| 🌙 **Dark/Light Mode** | Theme switcher with localStorage persistence |
| 📢 **Announcements Marquee** | Auto-scrolling news ticker |
| 💬 **WhatsApp Widget** | Floating pulsating WhatsApp chat button |
| 🎭 **Parallax Scrolling** | Depth effect on hero section background |
| ✅ **Form Validation** | Custom JS validation with shake animations |
| 📧 **Formspree Integration** | Email delivery for Admission & Contact forms |
| 🔍 **Scroll-Spy Navigation** | Auto-highlighting active nav links on scroll |
| 📄 **PDF Prospectus** | Downloadable school prospectus |
| 🖼️ **Lazy Loading** | Skeleton shimmer + native lazy load for images |
| 🎨 **Glassmorphism UI** | Frosted glass effects on cards and header |
| ⚙️ **SEO Optimized** | Open Graph tags for WhatsApp/Social link previews |
| 🎡 **Particles.js Background** | Animated connected dots background |

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure & SEO meta tags
- **CSS3** — Custom properties, Glassmorphism, Animations, Dark mode
- **Vanilla JavaScript** — IntersectionObserver, requestAnimationFrame, Fetch API
- **Particles.js** — Animated particle background
- **Vanilla Tilt.js** — 3D tilt effects on cards
- **Font Awesome 6** — Icon library
- **Google Fonts (Poppins)** — Typography
- **Formspree** — Serverless form submissions
